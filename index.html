Michael Mayoral published on 2020-11-16 included in PowerShell Security
Photo by Christina Morillo from Pexels If you’re like me you have many API integrations you’ve created, and for each of these you probably need to keep some kind of credential or secret key that needs to remain hidden. Furthermore, you probably have a bunch on these scripts running on scheduled tasks on some server. As you probably know, it is not a good idea to leave this key in clear text within your scripts.
